Examples for Schema 2.4

We have attempted to provide at least one example for each element. (There are also some examples for many of the attributes and types). The examples show minimal examples of how each element can be used and so are a guide to its semantics. By convention an element foo will have an example foo1.cml there may be further examples (foo2.cml etc.) which are often more complex.
